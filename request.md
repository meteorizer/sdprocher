# 프로세스 실행 상태 검사기
지정된 프로세스 목록에 대하여 현재 실행 상태를 점검하여 출력.

## 주요 사항

### 기본 사항
- python 2.7로 작성.
- psutil 패키지 사용
- rich한 출력이 가능한 경우 제공할 수 있도록.
- 단독 파일로 작성.
- 받은 프로세스 목록을 기준으로 하여 현재 실행 중인 프로세스의 상태를 점검한다.

### 입력
- csv, json 형식의 프로세스 목록 파일을 입력으로 받는다.
- 파이프 입력을 처리할 수 있다. (CSV, JSON )

#### 입력 항목

| key | 설명 | 필수 여부 |
| --- | --- | --- |
| run type | 프로세스 실행 타입 (python, java 등) | N |
| process name | 프로세스의 명칭 | Y |
| cmd | ps로 체크할 프로세스 문자열 | Y |
| path | 프로세스의 경로명 | N |


### 출력
- 프로세스 실행 여부를 출력
- 좀비 여부, 최초 실행 시각, 마지막 엑세스 시각을 포함
- 입력에 process name, description이 포함되어 있다면 출력에도 포함.

#### 출력 양식 예시

| 프로세스 타입 | 프로세스명 | 프로세스 상태 | 실행 시각 | 엑세스 시각 | cmd |
| --- | --- | --- |  --- | --- | --- |
| nginx | NginX | Y | 2026-02-22 15:00:20 | 2026-04-13 14:22:33 | nginx: master process /usr/sbin/nginx |
| python | 블랙홀 차단 처리 데몬 | N | 2026-04-04 01:11:20 | 2026-04-13 12:33:01 | python bblock.py --superman |
